<br>
Descrição do jogo:<br>
O jogador deve controla um artefato que se move horizontalmente na parte inferior da tela e deve derrotar ondas de invasores que se deslocam de lado a lado e descem gradualmente em direção ao artefato. O objetivo é eliminar todos os invasores antes que eles alcancem o jogador (artefato), usando um canhão laser do artefato para destruir as ordas inimigas enquanto se desvia dos tiros dos invasores.<br>
